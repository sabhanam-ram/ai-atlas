Recurring concepts correlated across every video.

  • Agents vs. Workflows

    1 mention(s) · updated run 1

    The architectural spectrum from predefined prompt chains (workflows) to a model run in a self-correcting loop (agents), and the hybrid 'workflows of agents' where each workflow step is itself a closed agent loop.

  • Context Engineering

    1 mention(s) · updated run 1

    Deliberately managing what occupies a model's finite context window — including offloading token-heavy subtasks to sub-agents that return only small results — to keep the working (short-term) memory focused. The working-memory layer of agent memory.

  • Model Context Protocol (MCP)

    1 mention(s) · updated run 1

    An open standard for connecting AI applications to external tools, data, and prompts through a uniform interface; the standard way to add custom tools to an agent loop.

  • Multi-Agent Systems

    1 mention(s) · updated run 1

    Architectures in which multiple model instances work concurrently — typically an orchestrator/parent agent delegating subtasks to parallel sub-agents — for parallelism, context isolation, and as a form of test-time compute.

  • Tool Use

    1 mention(s) · updated run 1

    The mechanism by which a model takes actions: it emits a structured call against a declared tool, the application executes it, and the result re-enters the context. The substrate beneath agents, sub-agents, retrieval, and MCP.
